## プログラム内容
入力した単語の母音の数を数えてくれるプログラム。
プログラムはmain.pyというファイルで作成した。

## クローンの仕方
$git clone https://github.com/HaruKato0915/robosys2025

## 実行方法
$cd robosys2025に移動
$echo "好きな英単語" | python3 main.pyで実行

## 実行結果
例えばhelloと入れると母音の数が2つより、2と出てくる

## テスト環境
Ubuntu22.04.5

## ソフトウェア
python
3.9~3.12まで対応
